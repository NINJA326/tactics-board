# Tactics Board Pro v17.1 Selected Share

## 追加修正
- 保存プレーを選択中の場合、画像保存・動画保存・コピー・共有・LINE送信の対象をその保存プレーに統一
- ムーブライブラリを選択中の場合も、同じく全共有機能の対象をそのライブラリプレーに統一
- 画像コピー/画像保存/共有は、選択中プレーの最終フレームをサムネイルとして出力
- 動画保存は、選択中プレー全体を録画
- LINEパネルに「選択中：プレー名（保存プレー/ムーブライブラリ）」を表示

## 使い方
1. 保存プレーまたはムーブライブラリを選択
2. 画像保存・動画保存・コピー・共有・LINEを押す
3. 選択中プレーがそのまま出力されます

## 注意
- MacではLINEへ完全自動添付はできないため、保存後にダウンロードからLINEへドラッグしてください。
- iPhone/Androidは共有シートにLINEが出れば直接送信できます。
