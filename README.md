## 追加修正
- 動画保存時のコート表示を、画面表示と同じPNGコート画像を使う方式に修正
- 選択中プレーのモード（ハーフ/オール）を動画保存へ正しく引き継ぎ
- 動画保存時にファイル名を入力できるように修正
- 保存プレー名を変更できるように修正
- ムーブライブラリ名を変更できるように修正
- iPhoneは動画保存後に「共有」→「ビデオを保存」で写真アプリへ保存可能
- 既存のムーブライブラリ、鍵ロック、画像保存、共有、ドライブ時ボール追従、再生機能は維持

## iPhoneで写真アプリへ入れる方法
1. 動画保存を押す
2. 名前を入力して保存
3. 共有を押す
4. 共有メニューから「ビデオを保存」を選択

※ブラウザの仕様上、自動で写真アプリへ直接保存はできません。
