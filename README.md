# Tactics Board Pro v16.0 Complete Release

GitHub Pagesへそのままアップロードできる販売版ベースです。

## アップロード方法
1. このZIPを解凍
2. 中のファイルを GitHub リポジトリ `tactics-board` のルートへ上書きアップロード
3. GitHub Pagesの公開URLを開く

公開URL例:
https://ninja326.github.io/tactics-board/

## 統合済み機能
- 販売版UI
- ハーフ / オールコート
- コート大型表示
- 初期ポジション保存
- 保存プレー
- ムーブライブラリ
- ライブラリ並び替え
- ロック / 名前変更 / 削除
- 移動 / ドライブ / パス / シュート / スクリーン
- スクリーンの進行方向側表示
- スペースゾーン追加 / 移動 / 伸縮 / 削除 / 番号管理
- 白丸番号から線種変更
- スマートDF
- 再生中・録画中のDF追従
- 再生速度 0.1〜1.5倍
- 画像保存
- 動画保存
- 黒画面対策済み録画エンジン
- サムネイル用タイトル表示
- 共有メニュー
- LINEは端末の共有シート経由
- サイドチェンジ表示切替

## 注意
- GitHub PagesなどHTTPS上で使うと動画保存・共有が安定します。
- iPhoneでLINEへ送る場合は、共有シートからLINEを選択してください。
- ブラウザ仕様上、LINEアプリを完全に直接指定することはできません。
