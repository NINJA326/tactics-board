# Tactics Board Pro v15.0 Pro Release

販売開始用の統合・安定版です。

## 統合内容
- 販売版UI / コート大きめ表示
- ハーフ / オール切替
- Dボタン下部配置
- 初期ポジション保存
- 保存プレー / ムーブライブラリ
- ライブラリのロック、名前変更、削除、ドラッグ並び替え
- 選択中プレー表示
- 移動 / ドライブ / パス / シュート / スクリーン
- 白丸番号から後で種類変更
- スクリーンマークを進行方向側にオフセット表示
- スペースゾーン追加、伸縮、番号管理、削除
- スマートDF AI
- 再生速度 0.1 / 0.25 / 0.5 / 0.75 / 1.0 / 1.25 / 1.5
- 画像保存 / 動画保存 / 共有
- 動画保存の黒画面・0秒対策
- プレー名入りサムネイル対応

## 使い方
`index.html` を開いて利用します。GitHub Pages にアップする場合は、このZIPの中身をリポジトリ直下へ配置してください。

## 注意
動画保存はブラウザ仕様の影響を受けます。GitHub PagesなどHTTPS環境での利用を推奨します。

## v15.1 追加修正
- 保存プレーとムーブライブラリに「↔」サイドチェンジボタンを追加
- 押すとプレーを左右鏡反転して複製
- 反転後の名前には「サイドチェンジ」を付与
- 選手、ボール、線、スクリーン、スペースゾーン、スペースゾーンの番号管理も反転対象
- 既存機能は変更なし


## v15.2 追加修正
- 保存プレー/ムーブライブラリのサイドチェンジを「複製保存」から「一時表示切替」に変更
- 保存数・ライブラリ数を増やさず、通常表示 ⇄ サイドチェンジ表示を切替可能
- 選択中プレーにも「↔ サイドチェンジ」ボタンを追加
- 画像保存・動画保存・共有は現在表示中の通常/サイドチェンジ状態を反映
- 既存機能は変更なし


## v15.3 追加修正
- サイドチェンジ時に保存名へ「サイドチェンジ」を付けない方式へ変更
- 選択中プレーは同じ名前のまま表示
- NORMAL / SIDE の小さなバッジで現在の向きを表示
- ボタンは「サイドチェンジ / 通常表示」の切替だけに整理
- 画像保存・動画保存・共有は現在の表示状態を使用

## 今後の改善案
- Undo / Redo：作成ミスをすぐ戻せる
- フォルダ管理：オフェンス、ディフェンス、BLOB、ATOなどに分類
- テンプレート：Horns、5-Out、Spain PnRなどをワンタップ配置
- QR共有：選手へURL/QRで即共有
